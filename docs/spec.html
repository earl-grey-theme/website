Prelude

Interpreting this specification

Language syntaxes and scopes will be referred to in the following manner throughout this document:

Scope :
( ForegroundColor , BackgroundColor ) Italic, Bold

… where BackgroundColor, Italic, and Bold are optional.

Throughout the document, broad ranges of syntax tokens will be referred to using TextMate Naming Conventions. (See also the Sublime Text documentation.)
